# 🌌 Neon Aura AR

Neon Aura AR is a browser-based hand tracking experience built using MediaPipe Hands, HTML5 Canvas, JavaScript, and the Web Audio API.

The project turns real-time hand movements into interactive neon visuals, gesture-based effects, procedural audio, and cinematic animations directly inside the browser.

## 🚀 Live Demo

👉 **Try it here:**
** https://pralhadt.github.io/neon-aura-ar./**

> Allow camera access when prompted to experience the hand tracking effects.

Everything runs completely on the client side with no backend required.

---

## ✨ Features

* Real-time hand tracking using MediaPipe Hands
* Supports up to 2 hands simultaneously
* Neon fingertip glow effects
* Particle trails and motion blur
* Electric lightning interactions between hands
* Gesture-triggered shockwaves
* Matrix-style animated background
* Multiple visual themes
* Procedural audio using Web Audio API
* FPS and gesture tracking HUD
* Real-time canvas rendering
